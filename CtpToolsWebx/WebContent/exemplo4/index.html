Domino AngularJS Toaster Notifications

This example will launch some toaster notifications which have been driven by some JSON data - the example URLS to call Domino Data has been commented out for the demo. Once the toaster notification has been cleared then the backend application will update the status of the corresponding notification document using a PATCH request.

For the demo purposes a list of cleared notifcations will be displayed instead.

The associated blog article which describes this code is available here.
